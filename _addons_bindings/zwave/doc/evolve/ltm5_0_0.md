# LTM-5 Wall Mount Accessory Switch
This describes the Z-Wave device *LTM-5*, manufactured by *Evolve* with the thing type UID of ```evolve_ltm5_00_000```.

The LTM-5 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

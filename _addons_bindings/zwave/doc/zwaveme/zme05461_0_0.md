# ZME\_05461 Double switch
This describes the Z-Wave device *ZME_05461*, manufactured by *Z-Wave.Me* with the thing type UID of ```zwaveme_zme05461_00_000```.

The device is in the category of *Wall Switch*, defining Any device attached to the wall that controls a binary status of something, for ex. a light switch.

The ZME_05461 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

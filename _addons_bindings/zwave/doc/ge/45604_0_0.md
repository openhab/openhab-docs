# 45604 Outdoor Lighting Control Module
This describes the Z-Wave device *45604*, manufactured by *Jasco Products* with the thing type UID of ```ge_45604_00_000```.

The 45604 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

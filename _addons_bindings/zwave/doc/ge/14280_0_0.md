# GE/Jasco 14280 GE/Jasco Z-Wave Plus Plug-In Dimmer Module
This describes the Z-Wave device *GE/Jasco 14280*, manufactured by *Jasco Products* with the thing type UID of ```ge_14280_00_000```.

The GE/Jasco 14280 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

### Dimmer

The brightness channel allows to control the brightness of a light.
            It is also possible to switch the light on and off.

The ```switch_dimmer``` channel supports the ```Dimmer``` item and is in the ```DimmableLight``` category.

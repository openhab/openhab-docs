# Reitz 05431 Duewi ZW BJ ES 1000 / Reitz 05431 / ZWave.me 05457 Single button wall switch
This describes the Z-Wave device *Reitz 05431*, manufactured by *Reitz-Group.de* with the thing type UID of ```reitz_05431_00_000```.

The Reitz 05431 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

# PSC152ZW EINBAU JAL|ROLL Z-WAVE
This describes the Z-Wave device *PSC152ZW*, manufactured by *permundo GmbH* with the thing type UID of ```permundo_psc152zw_00_000```.

The device is in the category of *Blinds*, defining Roller shutters, window blinds, etc..

![PSC152ZW product image](https://www.cd-jackson.com/zwave_device_uploads/826/826_default.png)


The PSC152ZW supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

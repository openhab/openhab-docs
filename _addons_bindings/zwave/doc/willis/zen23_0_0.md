# ZEN23 Wall mounted switch
This describes the Z-Wave device *ZEN23*, manufactured by *Willis Electric Co., Ltd.* with the thing type UID of ```willis_zen23_00_000```.

The ZEN23 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

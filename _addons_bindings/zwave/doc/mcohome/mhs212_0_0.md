# MH-S212 Mini Combo Switch
This describes the Z-Wave device *MH-S212*, manufactured by *[McoHome Technology Co., Ltd](http://www.mcohome.com/)* with the thing type UID of ```mcohome_mhs212_00_000```.

The MH-S212 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.

# HOWT01L Dewenwils Z-wave Smart Timer Box
This describes the Z-Wave device *HOWT01L*, manufactured by *Sigma Designs (Former Zensys)* with the thing type UID of ```sigma_howt01l_00_000```.

The device is in the category of *Battery*, defining Batteries, Energy Storages.

![HOWT01L product image](https://opensmarthouse.org/zwavedatabase/1695/image/)


The HOWT01L supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data between other devices in the mesh network.

## Overview

DEWENWILS 800 Series Z-Wave Long Range Timer, Heavy Duty 3HP, 240VAC, 40-Amp Metal Box Smart Switch for Pools, Pumps, Lights, AC Units, Electric Water Heaters, Z-Wave Hub Required, ETL Listed

### Inclusion Information

Press the power button on the smart timer box 3 times quickly.  The network LED will flash and the smart timer box will be in inclusion mode.   If inclusion was successful, network LED remains ON  

### Exclusion Information

Press the power button on the smart timer box 3 times quickly, the smart timer box will enter exclusion mode.  If exclusion was successful the network LED remains OFF

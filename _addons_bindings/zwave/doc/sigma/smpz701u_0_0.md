# SM-PZ701U Z-Wave power plug
This describes the Z-Wave device *SM-PZ701U*, manufactured by *Sigma Designs (Former Zensys)* with the thing type UID of ```sigma_smpz701u_00_000```.

![SM-PZ701U product image](https://www.cd-jackson.com/zwave_device_uploads/602/602_default.jpg)


The SM-PZ701U supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data from other devices.
